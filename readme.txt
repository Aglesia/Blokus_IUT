Bienvenue dans le projet de programmation du Blokus !!

le lancement du jeu s'effectue en executant le Blokus.jar présent dans ce dossier.

	-> Si java n'est pas installé sur votre Ordinateur, un fichier d'installation est disponible dans le dossier "Doc".

Le manuel d'utilisation du jeu se trouve lui-aussi dans le dossier "Doc".

Les dossiers "src", "javaDoc", "Ressources" et "class" sont utilisés pour la compilation du programme final (Blokus.jar).
